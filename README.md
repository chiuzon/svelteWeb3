## Installation
Be sure to have installed `buffer` `@web3-react/types` and an `@web3-react` connector 

```
    yarn add @chiuzon/svelteweb3 -D
    yarn dev
```

## Local Development

- Clone repo\
`https://github.com/chiuzon/svelteWeb3`

- Install dependencies\
`yarn`

- Build and watch for changes\
`yarn dev`
